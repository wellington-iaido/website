All Japan Kendō Federation Iaidō

全日本剣道連盟居合道

Zen Nihon Kendō Renmei Iaidō

The All Japan Kendō Federation is an umbrella organisation created after the war that brings together various weapon based martial arts including kendō, jodō and iaidō.

In 1967 a group of the top sensei saw a need for a common set of iaidō kata to allow greater cooperation between the different schools. In this representative set of forms called seitei gata, each kata is a modified version of an older one and the set explores a range of traditional iaidō schools or koryū.

To an original seven kata, a further three were added in 1980, and two more in 2002 to round out a set of twelve kata making up seitei gata iaidō. Seitei gata is practiced throughout the world as a beginner set and is used for grading and competitions.
